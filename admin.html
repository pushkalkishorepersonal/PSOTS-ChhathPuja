📲 Fonnte WhatsApp API
Fonnte powers automated WhatsApp messages — payment confirmations, broadcast blasts, and payment reminders — without you having to manually open WhatsApp.
⏳ Not checked yet
Setup (one-time, done in Apps Script):
1. Go to script.google.com → open your project → Project Settings → Script Properties
2. Add a property: Name = FONNTE_TOKEN  Value = your Fonnte API token
3. Get your token at fonnte.com → add device → scan QR with your WhatsApp → copy the token
4. Run authorizeExternalRequests() once to grant HTTP permission
